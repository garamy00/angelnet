UpNote

  • UpNote 의 note/new 의 notebook 파라미터에 전달할 노트북 이름.
  • 하위 노트북은 슬래시(/) 로 경로 표기 (예: 업무/2026).
  • 일일/주간 노트(상단) — 본인 기록용. 일일업무보고 페이지의 🔄 UpNote 저장(주) 이 사용.
  • 주간업무보고(하단) — 팀장 보고용. 주간업무보고 페이지의 📤 UpNote 저장 이 사용.
  • 비워두면 해당 액션 비활성화.
  • 주간 제목 변수: {{ yyyy }}, {{ yy }}, {{ mm }}, {{ ww }} (ISO 주차), {{ ww_of_month }} (월의 몇 번째 주), {{ week_start_mmdd }}, {{ week_end_mmdd }}.
  • 체크하면 본문을 ``` 로 감싸 markdown 자동 변환을 비활성화합니다.
  • 차단되는 변환: 들여쓰기 제거, * → bullet, 자동 번호 등.
  • plain text 형식을 정확히 보존하려면 체크.
  • UpNote 의 markdown 파라미터.
  • 일부 버전은 false 라도 자체 변환이 적용 — 결과가 어색하면 위 "코드 블록으로 감싸기" 도 체크.

출력 템플릿

  • Jinja2 문법.
  • 사용 가능 변수: {{ entry.category }}, {{ entry.body }}, {{ yy }}, {{ ww }}, {{ days }}, {{ week_notes }} 등.
