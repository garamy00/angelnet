UpNote

UpNote 의 note/new 의 notebook 파라미터에 전달할 노트북 이름. 하위 노트북은 슬래시(/) 로 경로 표기 (예: 업무/2026). 일일/주간 노트(상단)는 본인 기록용 — 일일업무보고 페이지의 🔄 UpNote 저장(주) 가 사용. 주간업무보고(하단)는 팀장 보고용 — 주간업무보고 페이지의 📤 UpNote 저장 이 사용. 비워두면 해당 액션 비활성화.

체크하면 본문을 ``` 로 감싸 markdown 자동 변환 (들여쓰기 제거, * → bullet, 자동 번호 등) 을 모두 비활성화합니다. plain text 형식을 정확히 보존하려면 체크.

UpNote 의 markdown 파라미터. 일부 버전에서는 false 라도 자체 변환이 적용될 수 있으니, 결과가 어색하면 위 "코드 블록으로 감싸기" 도 체크하세요.

출력 템플릿

Jinja2 문법. {{ entry.category }}, {{ entry.body }}, {{ yy }}, {{ ww }}, {{ days }}, {{ week_notes }} 등.
